== Description ==

This Wordpress plugin uses several different tools to create a news ticker with your latest tweets being displayed.

== Installation ==

1. Unzip and upload `wp-twitter-ticker-widget` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place WP Twitter Ticker widget in a widget area and configure.
